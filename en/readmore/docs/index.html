Documentation

ReadMore.js is a lightweight and flexible JavaScript plugin for creating convenient expandable text blocks with «Read More» and «Hide» buttons. It optimizes the display of large amounts of content, improving readability and saving space on the page. The plugin is ideal for websites, blogs, news portals, and other projects where managing long texts is necessary, both on desktops and mobile devices.

Note
  • The plugin operates standalone and does not require external libraries such as jQuery.
  • All content remains accessible to search engines, as the plugin does not alter the HTML structure of the page.

Easy Integration

Easily integrates into any project with a single JavaScript file. Simply include it and configure it to suit your needs.

Flexible Configuration

The plugin offers a wide range of parameters: from the height of the collapsed block to button text, animation speed, and even custom callback functions for event handling.

Responsive Design

Supports responsive breakpoints, allowing you to customize the plugin's behavior based on screen width for optimal display across all devices.

SEO and Accessibility

Content remains accessible to search engines, and support for accessibility attributes enhances usability for people with disabilities, improving overall site accessibility.
